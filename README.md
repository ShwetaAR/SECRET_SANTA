# SECRET_SANTA

## What is Secret Santa Game

Secret Santa is a Western Christmas tradition in which members of a group or community are randomly assigned a person to whom they give a gift.

### How to start the game 
- Click on clone or download
- Clone the project into your ide
- import the project from your local git repository 
- import as existing project
- finish
- Go to SecretSantaLauncher class to run the program.
- Right click run as java application
- Now follow the instructions.

File will be created for new team members.
